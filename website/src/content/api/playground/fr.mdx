Exécutez des requêtes sur l'endpoint GraphQL en production à l'adresse `https://api.rolebase.io/graphql`. Ouvrez l'onglet **Headers** et renseignez votre clé API pour charger le schéma et lancer des requêtes authentifiées.

```json
{ "x-api-key": "VOTRE_CLE_API_ICI" }
```

[Générez une clé API](/apps) depuis les paramètres de votre organisation. Consultez les [Intégrations personnalisées](/fr/developers/custom-integrations) pour le fonctionnement de l'authentification, et l'[Introduction](/fr/api) pour la référence complète.
